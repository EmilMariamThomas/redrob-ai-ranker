# 🧠 RedRob AI Candidate Ranking Engine

## 🚀 Overview
A CPU-optimized AI-powered candidate ranking system that processes 100,000+ profiles and ranks them using a multi-factor scoring engine.

---

## ⚙️ Features

- 📊 Scales to 100,000 candidates
- 🧠 Multi-dimensional scoring:
  - Technical skills
  - Career background
  - Behavioral signals
- 🧾 Explainable AI (WHY each candidate is ranked)
- ⚡ CPU-only, no ML model dependency
- 📦 Generates Top 100 submission file

---

## 🧠 Ranking Logic

Final Score =  
40% Technical Skills  
30% Career Experience  
30% Behavioral Signals  

---

## 🔍 Explainability

Each candidate includes:
- Why they are ranked
- Skill strength analysis
- Behavioral readiness
